# Prêmio Pintor ABRAPP Jornal do Pintor - Interface Web

Este projeto replica a interface do sistema Python (Streamlit) usando HTML5, CSS3 (Bootstrap), JavaScript com PropTypes e gráficos ECharts via CDN.

## Estrutura
- `index.html`: Página principal
- `styles.css`: Estilos customizados
- `js/`: Scripts JavaScript (componentes reutilizáveis)
- `charts/`: Gráficos ECharts

## Como usar
Abra o arquivo `index.html` em seu navegador. Não é necessário instalar dependências.

## Responsividade
A interface é totalmente responsiva, funcionando em desktop, tablet e mobile.

## Exemplos de gráficos
Os gráficos replicam os principais charts do projeto Python, usando ECharts.

## Customização
- Adicione novos charts em `charts/`
- Componentes reutilizáveis em `js/`
